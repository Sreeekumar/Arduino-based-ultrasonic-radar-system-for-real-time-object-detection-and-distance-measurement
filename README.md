# Arduino-Based Ultrasonic Radar System

## Overview
An Arduino-based system designed for real-time object detection and distance measurement using an ultrasonic sensor and time-of-flight principle.

## Features
- Real-time distance measurement  
- Object detection using ultrasonic sensing  
- Simple and low-cost implementation  

## Components / Technologies
- Arduino Uno  
- Ultrasonic Sensor (HC-SR04)  
- Breadboard  
- Jumper Wires  

## Working
The ultrasonic sensor emits sound waves → receives echo → Arduino calculates distance using time-of-flight → object presence is detected.
